# node-basic-crud
Node Js basic crud operation for User using express-handlebars 


Steps:
 (1) git clone https://github.com/imjaipatel248/node-basic-crud.git
 
 (2) cd node-basic-crud
 
 (3) npm install
 
 (4) make .env with keys:
      DB_HOST = xxxxxxx
      DB_NAME = mode_yt
      DB_USER = xxxx
      DB_PASS = xxxx
      
 (5) npm start
